# 📄 PDF 만능 도구

병합 · 분할 · 압축 · 이미지 변환 · 페이지 정리 · 쪽번호 · 워터마크 · 모아찍기 · 텍스트 추출 · OCR 을
**설치 없이, 무료로, 무제한** 사용하는 웹 도구입니다.

> 제작: **해누리중학교 조승재** · chuseonjae@outlook.kr

## ✨ 기능

| | 기능 | 설명 |
|--|------|------|
| 🔗 | 병합 | 여러 PDF를 하나로, 순서 조정 |
| ✂️ | 분할 | 페이지 범위 추출 / 낱장 분리 (ZIP 저장) |
| 🗜️ | 압축 | 용량 줄이기 + 흑백 변환 |
| 🖼️ | 이미지→PDF | JPG·PNG·WEBP 여러 장을 PDF로 |
| 📷 | PDF→이미지 | 각 페이지를 PNG/JPG로 (ZIP 저장) |
| 📑 | 페이지 정리 | 미리보기로 회전·삭제·순서 변경 |
| 🔢 | 쪽번호 | 위치·모양·시작번호 선택 |
| 💧 | 워터마크 | 한글 글자 또는 로고 이미지 |
| 🖨️ | 모아찍기 | 2·4쪽 모아찍기, 소책자 |
| 📝 | 텍스트 추출 | PDF 글자 복사·TXT 저장 |
| 🔍 | OCR | 스캔본·사진의 글자 인식 (한국어·영어) |

## 🔒 개인정보

모든 처리는 **여러분의 브라우저 안에서만** 이루어집니다. 파일이 서버로 전송되지 않으므로
학생 자료·시험지 등 민감한 문서도 안전합니다.

## 🚀 사용 방법

- **온라인**: 배포된 주소를 브라우저에서 열기만 하면 됩니다. (PC·태블릿·휴대폰)
- **내 컴퓨터에서**: `PDF도구 실행.bat` 더블클릭 (Python 필요) — 인터넷 없이도 동작

## 🧩 사용한 오픈소스

- [pdf-lib](https://github.com/Hopding/pdf-lib) (MIT) — PDF 생성·편집
- [pdf.js](https://github.com/mozilla/pdf.js) (Apache-2.0) — PDF 렌더링
- [tesseract.js](https://github.com/naptha/tesseract.js) (Apache-2.0) — OCR 엔진
- [fflate](https://github.com/101arrowz/fflate) (MIT) — ZIP 생성

한글 워터마크 글자는 브라우저에 설치된 시스템 글꼴을 이용해 이미지로 그리므로,
별도의 글꼴 파일을 포함하거나 재배포하지 않습니다.
