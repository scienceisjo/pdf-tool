여러 PDF를 하나로 합치기

파일을 여러 개 올리고 순서를 조정한 뒤 하나의 PDF로 병합합니다.

클릭하거나 파일을 여기로 끌어다 놓으세요 여러 개 선택 가능 · PDF 파일

PDF를 나누기

페이지를 미리 보면서 드래그로 범위를 선택하거나, 페이지 사이 가위(✂)로 잘라 나눕니다.

클릭하거나 파일을 여기로 끌어다 놓으세요 PDF 파일 1개
· 페이지를 클릭(터치는 톡 누르기)하면 선택돼요. 마우스로 드래그하면 여러 쪽이 한꺼번에 선택돼요(파란색). · 페이지 사이 ✂를 누르면 그 자리에서 잘려요(색깔 구간으로 나뉨).
⚡ 빠른 도구
번호로 선택
몇 쪽씩 자르기 쪽마다 ✂
용량으로 나누기 MB 이하로
‘번호로 선택’·‘몇 쪽씩’은 화면에 표시만 해요. 실제 저장은 아래 파란/초록 버튼을 누르세요. ‘용량으로 나누기’는 바로 저장돼요(이메일 첨부 용량 제한용).

PDF 용량 줄이기

이미지가 많거나 스캔한 PDF의 용량을 크게 줄입니다. 흑백 변환으로 잉크도 아껴요.

클릭하거나 파일을 여기로 끌어다 놓으세요 PDF 파일 1개
※ 스캔본·사진 PDF에서 효과가 큽니다. 글자만 있는 PDF는 거의 안 줄어들 수 있어요.
미리보기 (이 설정으로 변환한 첫 페이지)

이미지를 PDF로 변환

사진·그림(JPG, PNG 등)을 여러 장 올리면 한 장씩 페이지로 담은 PDF를 만듭니다.

클릭하거나 이미지를 여기로 끌어다 놓으세요 여러 장 선택 가능 · JPG, PNG, WEBP, BMP, GIF
이미지는 용지 안에 비율을 유지한 채 최대 크기로, 가운데에 배치됩니다.
미리보기 (첫 번째 이미지가 놓이는 모습)

PDF를 이미지로 변환

PDF의 각 페이지를 그림 파일(PNG/JPG)로 저장합니다. 수업자료·발표에 붙여넣기 좋아요.

클릭하거나 파일을 여기로 끌어다 놓으세요 PDF 파일 1개

페이지 정리 (회전 · 삭제 · 순서 바꾸기)

페이지를 미리 보면서 회전하고, 지우고, 순서를 바꾼 뒤 새 PDF로 저장합니다.

클릭하거나 파일을 여기로 끌어다 놓으세요 PDF 파일 1개
각 페이지에서 ◀ ▶ 순서 이동 · ↻ 90°씩 회전 · ✕ 삭제(다시 누르면 복구)

쪽번호 매기기

모든 페이지에 자동으로 쪽번호를 찍어줍니다. 학습지 묶음·자료집 만들 때 좋아요.

클릭하거나 파일을 여기로 끌어다 놓으세요 PDF 파일 1개
미리보기 (첫 페이지)

워터마크 넣기

모든 페이지에 반투명 글자나 로고 이미지를 넣습니다. (예: "배포용", 학교 로고)

클릭하거나 파일을 여기로 끌어다 놓으세요 PDF 파일 1개
클릭해서 이미지 선택 JPG, PNG (투명 배경 가능)
미리보기 (첫 페이지)

모아찍기 · 소책자

여러 쪽을 한 장에 모아 인쇄 용지를 아끼거나, 반으로 접으면 책이 되는 소책자를 만듭니다.

클릭하거나 파일을 여기로 끌어다 놓으세요 PDF 파일 1개
미리보기 (한 장에 어떻게 배치되는지)

텍스트 추출

PDF 안의 글자를 뽑아냅니다. 복사해서 한글·워드에 붙여넣거나 TXT로 저장하세요.

클릭하거나 파일을 여기로 끌어다 놓으세요 PDF 파일 1개 (스캔 이미지 PDF는 글자 추출이 안 될 수 있어요)

OCR — 스캔본 글자 인식

스캔한 PDF나 사진 속 글자를 인식해서 텍스트로 만들어 줍니다. 한국어·영어 지원, 100% 오프라인.

클릭하거나 파일을 여기로 끌어다 놓으세요 PDF 1개 또는 사진 여러 장 (JPG, PNG)
※ 처음 실행할 땐 인식 엔진을 준비하느라 10~20초쯤 걸리고, 이후 페이지당 수 초~수십 초 걸려요. 글자가 선명할수록 정확합니다.
